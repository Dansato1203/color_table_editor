# Color table editor

## Installation

```shell
git clone https://github.com/satoshishimada/color_table_editor.git
cd color_table_editor
make
```

## Usage

### Make image list

```shell
ls image_dir/*.jpg | sort > image_list
```

### Key binds

| Key | Function |
| :-: | :-: |
| ESC | Exit |
| l | Load color table |
| s | Save color table |
| r | Reset color table |
| n | Next image |
| p | Previous image |
| 0 - 9 | Color margin |
| g | Select label green |
| w | Select label white |
| b | Select label black |
